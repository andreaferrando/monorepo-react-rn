# expo-monorepo
An expo monorepo boilerplate using lerna and yarn workspaces for creating react native apps. 

## Prerequisites
You need to install [watchman](https://facebook.github.io/watchman/)

## Installation
1. Run `yarn`
2. Run `yarn start`
